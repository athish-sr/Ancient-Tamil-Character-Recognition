# Ancient Tamil Character Recognition

This project contains preprocessing and character-segmentation scripts for ancient Tamil inscription images.

## Main Pipeline

The primary script is:

- `wavelet.py`

It combines:

1. Preprocessing (W1):
- Load image
- Grayscale conversion
- Wavelet denoising
- CLAHE enhancement

2. Character detection (W2):
- Binarization
- Contour filtering
- Green bounding-box drawing
- Character crop extraction

## Folder Structure (Important)

Input folder:

- `test_images/`

Main output folder (generated by `wavelet.py`):

- `wavelet_output/`

For each input image, outputs are saved in a separate subfolder with files like:

- `01_original.jpg`
- `02_grayscale.jpg`
- `03_wavelet.jpg`
- `04_clahe.jpg`
- `binary_for_detection.jpg`
- `clahe_bounding_boxes.jpg`
- `characters/char_*.png`

## Setup

1. Create and activate a virtual environment (recommended).
2. Install dependencies:

```bash
pip install -r requirement.txt
```

## Run

```bash
python wavelet.py
```

## Notes

- Bounding boxes are drawn in green.
- The script clears and recreates `wavelet_output/` on each run.
- Additional experimental scripts are also present in the repository (`w1.py`, `w2.py`, `4.py`, `5.py`, etc.).
